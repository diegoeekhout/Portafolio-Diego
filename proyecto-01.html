Ingresa medicamentos

Descripción

Es una aplicación web con un diseño minimalista encargada de agregar, editar y eliminar medicamentos de una farmacia. Se le dio prioridad al funcionamiento y no a sobrecargar la mima con mucho apartado visual, ya que lo importante era trabajar con la base de datos. El lenguaje principal que fue utilizado es JSP, con patrón de diseño DAO, se implemento TomCat la base de datos esta escrita en MySQL, el apartado visual con Boostrap.

Lenguajes Utilizados

  • JSP
  • DAO
  • TomCat
  • CSS3
  • Bootstrap
  • MySQL
